# Fuentes de Datos

## Datos incluidos en el repositorio
- `datos_macro_banxico.xlsx` — TC, INPC, IGAE descargados via API BANXICO
- `datos_clima_michoacan.xlsx` — Variables climaticas procesadas de CONAGUA/SMN

## Datos NO incluidos (descargar manualmente)

### SNIIM — Precios mayoristas
- URL: http://www.economia-sniim.gob.mx
- Archivo: `Base_Limon_SNIIM_RoxP.xlsx`
- Hojas: mexicano, persa, validacion_2026

### SIAP — Produccion agricola
- URL: https://nube.siap.gob.mx/cierre_agricola
- Archivos: `Cierre_agricola_mun_YYYY.csv` (2011-2024)

### CRE — Precio del diesel
- URL: https://www.datos.gob.mx/dataset/historico_precios_gasolinas_y_diesel
- Archivos: `historico_precios_expendio_YYYY.csv` (2017-2025)

### CONAGUA/SMN — Clima
- URL: https://smn.conagua.gob.mx
- Estaciones: Paracuaro (16085), Apatzingan (16007)
- Archivos: `climaParacuaro.txt`, `climaApatzingan.txt`
