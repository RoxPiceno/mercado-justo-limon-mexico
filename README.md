# Mercado Justo — Analisis y Prediccion de Precios de Limon en Mexico

## Descripcion
Proyecto academico de Ciencias de Datos para Negocios (5 semestre, 2026-1).
Analisis integral del mercado del limon mexicano con el objetivo de reducir
la brecha de intermediacion entre productores y mayoristas.

## Equipo
- **Alberto** - Regresion lineal simple
- **Gretel** - Cadena de Markov + ARIMA anual
- **Rox** - Machine Learning completo (13 modelos, analisis mensual)
- **Axel** - Investigacion de Operaciones

## Hallazgo principal
El productor de limon recibe en promedio el **42.7%** del precio mayorista.
Esta brecha es estructural y estable durante 14 anos (2011-2024).

## Fuentes de datos
- SNIIM - precios mayoristas
- SIAP - produccion agricola por estado
- BANXICO - tipo de cambio, INPC, IGAE
- CRE/datos.gob.mx - precio del diesel
- CONAGUA/SMN - variables climaticas Michoacan
